# Описание проекта
Графическое приложение для конвертации валют с использованием внешнего API.  
Программа позволяет пользователю выбрать валюты, ввести сумму и получить результат конвертации по актуальному курсу.

История всех операций сохраняется в файл `history.json` и автоматически загружается при запуске.

---

# Функциональность

 Выбор валют (из / в)  
 Ввод суммы  
 Получение актуального курса через API   Отображение результата  
  Сохранение истории в JSON  
  Загрузка истории при запуске  
  Таблица с историей операций  
  Валидация пользовательского ввода
